# Jarvis-Self-Assistant
User Guide for Jarvis :-
1. say "hey Jarvis" - It open a small window , where you saw four modules.
2. To launch module say "launch and module name " , Ex:- "launch voice command"
3. say "stop or exit" to stop the module.
4. your command must be clear .
5. Bychance it not respnding through voice , use buttons to launch modules.
